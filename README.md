## Installation

### 1. Clone or download this repository

```bash
git clone https://github.com/nikhilgangrade/Personal-website.git
cd Personal-website
```

### 2. Install requirements

```bash
pip install flask
```

### 3. Run the App

```bash
python myapp.py
```
By default, it runs on http://127.0.0.1:5000
